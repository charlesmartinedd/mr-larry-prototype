# Mr. Larry's Financial Literacy — 360° Interactive eLearning Prototype

A comprehensive showcase of 16 eLearning interaction types overlaid on a draggable 360° panorama image. Built as a single HTML file with zero dependencies.

## Live Demo
[View on GitHub Pages](https://charlesmartinedd.github.io/mr-larry-prototype/)

## Interaction Types

| Category | Color | Interactions |
|----------|-------|-------------|
| **Content Display** | Gold | Info Panel, Image Gallery, Video Embed, Audio Player, PDF Viewer |
| **Assessment** | Coral | Multiple Choice, Multiple Select, True/False, Drag & Drop, Fill in the Blank, Likert Survey |
| **Interactive Tools** | Teal | Compound Interest Calculator, Timeline, Flashcards |
| **Gamification** | Purple | Scenario Branching, Timed Challenge |

## Features
- Draggable 360° panorama (Poly Haven HDR)
- 16 color-coded interactive hotspots
- Progress tracking HUD (0/16 → 16/16)
- Fully responsive overlay panels
- Zero external dependencies — single HTML file
